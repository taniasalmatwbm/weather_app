03d  → ☁️ Scattered Clouds

04d  → ☁️ Broken Clouds

09d  → 🌧 Shower Rain

10d  → 🌦 Rain

11d  → ⛈ Thunderstorm

13d  → ❄ Snow

50d  → 🌫 Mist

if(icon === "01d"){
   // Sun Animation

if(icon === "09d"){
   // Rain Animation
}

   // Thunder Animation
